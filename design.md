# Markdown Editor - モバイルアプリ設計

## 設計方針

- **対象**: iOS / Android（ポートレート向き、9:16）
- **使いやすさ重視**: シンプルで直感的なUI
- **一手操作対応**: 片手での操作を想定した設計

---

## スクリーン一覧

### 1. **ホーム画面（ファイル一覧）**
- 作成済みのMarkdownファイル一覧を表示
- 各ファイルのプレビュー（最初の数行）
- ファイル作成日時、更新日時を表示
- 新規作成ボタン（フローティングアクションボタン）

### 2. **エディタ画面**
- **左側**: Markdownテキスト入力エリア
- **右側**: リアルタイムプレビュー（スワイプで切り替え可）
- トップバー: ファイル名、保存状態表示
- ボトムバー: 保存、ダウンロード、共有ボタン

### 3. **ファイル管理画面**
- ファイルの一覧表示（グリッド/リスト切り替え）
- 検索機能
- ファイルの削除、名前変更、移動機能
- クラウド同期状態の表示

### 4. **クラウド同期設定画面**
- iCloud / Google Drive との連携設定
- 同期状態の確認
- 同期ファイルの選択

### 5. **テーマ・設定画面**
- ダークモード / ライトモード切り替え
- フォントサイズ調整
- その他の設定（言語、通知など）

### 6. **ダウンロード画面**
- ダウンロード履歴表示
- ダウンロード形式選択（Markdown、PDF、HTML）
- ダウンロード先の選択

---

## 主要ユーザーフロー

### フロー1: 新規ファイル作成
1. ホーム画面で「新規作成」ボタンをタップ
2. ファイル名入力ダイアログが表示
3. エディタ画面に遷移
4. Markdownを入力 → リアルタイムプレビュー更新
5. 「保存」ボタンをタップ → ホーム画面に戻る

### フロー2: ファイル編集
1. ホーム画面でファイルをタップ
2. エディタ画面で編集
3. 自動保存（または手動保存）
4. 「戻る」で ホーム画面に戻る

### フロー3: ファイルダウンロード
1. エディタ画面で「ダウンロード」ボタンをタップ
2. ダウンロード形式を選択（Markdown / PDF / HTML）
3. デバイスのダウンロードフォルダに保存
4. 確認メッセージ表示

### フロー4: クラウド同期
1. 設定画面で「クラウド同期」を選択
2. iCloud / Google Drive を選択
3. 認証画面で認証
4. ファイルの同期開始
5. 同期完了メッセージ表示

## 色選定

### ブランドカラー
- **プライマリ**: `#0a7ea4`（落ち着いた青）
- **背景（ライト）**: `#ffffff`（白）
- **背景（ダーク）**: `#151718`（濃いグレー）
- **テキスト（ライト）**: `#11181C`（濃いグレー）
- **テキスト（ダーク）**: `#ECEDEE`（薄いグレー）
- **アクセント**: `#22C55E`（緑：保存成功）
- **警告**: `#EF4444`（赤：エラー）

## レイアウト仕様

### ホーム画面
- トップバー: アプリタイトル、設定ボタン
- コンテンツ: ファイル一覧（カード形式）
- ボトムバー: 新規作成ボタン（フローティング）

### エディタ画面
- トップバー: ファイル名、保存状態
- コンテンツ: エディタ + プレビュー（タブ切り替え）
- ボトムバー: 保存、ダウンロード、共有ボタン

### 設定画面
- トップバー: 「設定」タイトル
- コンテンツ: 設定項目リスト
- 各項目: トグル、スライダー、ボタンなど

---

## 主要機能

| 機能 | 説明 |
|------|------|
| **Markdown編集** | リアルタイムプレビュー付きエディタ |
| **ファイル管理** | 作成、編集、削除、名前変更 |
| **クラウド同期** | iCloud / Google Drive 連携 |
| **ダウンロード** | Markdown / PDF / HTML 形式でエクスポート |
| **テーマ** | ライト / ダークモード切り替え |
| **自動保存** | 編集内容を自動保存 |
| **検索** | ファイル名、内容で検索 |

---

## 技術スタック

- **フレームワーク**: React Native (Expo)
- **言語**: TypeScript
- **スタイリング**: NativeWind (Tailwind CSS)
- **状態管理**: React Context + AsyncStorage
- **クラウド同期**: iCloud (iOS) / Google Drive (Android)
- **ダウンロード**: expo-file-system, expo-sharing
- **Markdown処理**: react-markdown, remark
